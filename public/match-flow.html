API Endpoint Map Backend
All endpoints called during match journeys
Method Endpoint Called By Purpose Side Effects
POST /api/match/create Person A (both paths) Create match request. If partner_schedule included → auto-complete Creates DB row
GET /api/match/:token Person B init, Person A polling Check status, get schedules when completed None
POST /api/match/:token/complete Person B Submit Person B's schedule, mark match completed Email to Person A via Resend
GET /api/match/suggestions Result page Fetch date night ideas for overlap days None
POST /api/match/invite Result page invite sheet Create date invite with opportunity + message Creates DB row
GET /api/match/invite/:token date-invite.html Load invite details for recipient None
POST /api/match/invite/:token/respond date-invite.html Accept or decline date invite Email to sender via Resend
POST /api/auth/request Result signup, date-invite signup Create account + magic link Creates user, returns magic token
